Definition:

Server: A system (hardware and/or software) that provides resources, services, or data to other computers (clients) over a network using defined protocols.
Key characteristics:

Role: Responds to client requests.
Form: Can be a dedicated physical machine, a virtual machine, a container, or just server software running on a device.
Protocols: Common ones include HTTP/HTTPS, TCP, UDP, SMTP, FTP, SSH, DNS.
Common types of servers (brief):

Web server: Serves web pages (e.g., Apache, Nginx).
Application server: Runs backend application logic (e.g., Node.js, Tomcat).
Database server: Stores and serves data (e.g., MySQL, PostgreSQL).
File server: Stores and shares files (e.g., SMB/Samba, NFS).
Mail server: Sends/receives email (e.g., Postfix, Exchange).
DNS server: Resolves domain names to IPs (e.g., BIND).
Proxy server: Forwards requests, can cache or filter (e.g., Squid, HAProxy).
Cache/CDN server: Caches content for faster delivery (e.g., Varnish, Cloud CDN).
Authentication server: Handles logins and identity (e.g., LDAP, Active Directory).
Media/Streaming server: Streams audio/video (e.g., Wowza, Icecast).
Game server: Hosts multiplayer game sessions.
Virtualization/cloud server: Provides virtual machines or cloud instances (e.g., VMware, AWS EC2).
